# Data Availability Statement v1

The analysis uses public-access oral-history and registry materials distributed by Densho. The public release repository is available at https://github.com/pp33pant/oral-history-replication-public and the archived Zenodo record for the current frozen release is https://zenodo.org/records/19716590. The current version DOI is https://doi.org/10.5281/zenodo.19716590, while the concept DOI for the release series is https://doi.org/10.5281/zenodo.19716589. The replication package included with the release exposes the shareable processed research frame required to reproduce the paper-facing model tables, weighting summaries, and robustness summaries, along with the scripts, codebook documents, and appendix notes needed to audit the frozen measurement and modeling contract.

Direct transcript payloads are not redistributed in the replication package. Replicators who need the underlying transcript text should obtain it from Densho under Densho's public-access terms and then rebuild that layer from the documented pipeline. The released package therefore supports reproduction of the paper's frozen causal and robustness outputs from the processed research frame, while routing direct source-text access back to the original archive.

Repository code is released under the MIT License. The packaged derived data and frozen numeric outputs are distributed as part of the project replication archive. The archive manifest records the delivered package hash for audit purposes, and the version DOI should be used when citing this exact release.
